Detailed Description

The TestRunnerApi retrieves and runs tests programmatically from code inside the project, or inside other packages.

TestRunnerApi is a ScriptableObject. You can initialize the API like this:

var testRunnerApi = ScriptableObject.CreateInstance&lt;TestRunnerApi&gt;();

Note: You can subscribe and receive test results in one instance of the API, even if the run starts from another instance. The TestRunnerApi supports the following workflows:

  • How to run tests programmatically
  • How to get test results
  • How to retrieve the list of tests
